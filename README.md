# KawaMood's Waystones in Villages
[MeiaQuatru](https://modrinth.com/user/MeiaQuatru) originally wrote a datapack titled [KawaMood's Waystones in Villages ](https://modrinth.com/datapack/waystones-in-villages).  
The last working version of this datapack was for Minecraft Java Edition 1.21.1 and is now an archived project.  

This fork makes a small change to have it work with Minecraft Java Edition 1.21.9 and possibly later.  

Thank you to MeiaQuatru for letting me do this!
